# LizLang
LizLang is an experiment made by me to practice making a complex c-style programming language. It's heavily inspired by 
Kotlin, C#, Java, and C. The project is a multi-module project containing the following modules:
* A parser module which contains code
* An interpreter which contains code for running the language
* A compiler for compiling the code to either machine
* An Intellij Plugin for syntax highlighting and error detection
